# React-Management-Tutorial

PostgreSQL와 Node.js를 활용한 React App 입니다.

고객 관리 시스템으로 번호, 프로필 이미지, 이름, 생년월일, 성별, 직업 데이터를 처리합니다.

고객을 추가하고 삭제할 수 있습니다.

아래 출처의 강의에서 데이터베이스를 MySQL에서 PostgreSQL로 교체했습니다. 그리고 host는 localhost로 진행했습니다.
